# üêß Linux Process Monitoring Commands

## ‚úÖ `top`
Real-time system monitoring tool.  
Shows CPU usage, memory usage, load average, and running processes.

---

## ‚úÖ `ps`
Displays currently running processes on the system.  
Useful to check what programs are active.

---

## ‚úÖ `ps -a`
Shows processes of all users that are attached to a terminal.  
Helpful for viewing active user sessions and their processes.

---

## ‚úÖ `ps aux` ‚≠ê (Important for Interviews)
Displays all running processes in detailed format, including:

- User
- PID
- CPU usage
- Memory usage
- Start time
- Command

Commonly used for troubleshooting and process audits.

---

## ‚úÖ `htop`
Advanced and interactive version of `top`.

Features:
- Colorful UI
- Easy process search
- Kill processes interactively
- Better CPU & memory visualization

---

## ‚úÖ Process Filtering Examples

### `ps u`
Shows processes in a user-oriented format with details like CPU and memory usage.

### `ps aux | grep ping`
Used to find a specific process (e.g., `ping`).
